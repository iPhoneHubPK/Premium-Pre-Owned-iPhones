How to Update Your Live Website
  1. Make changes in any section of this admin
  2. Click Save All (top right) — saves to browser memory
  3. Go to Export JSON → click Download data.json
  4. Open your GitHub repo → click data.json → ✏️ Edit → paste new content → Commit
  5. Website updates in ~30 seconds ✅
⚠ Save All saves to your browser only. Always Export JSON to make changes permanent.

GitHub Upload — Step by Step
  1. Download data.json above
  2. Open your GitHub repository in a browser
  3. Click on the data.json file
  4. Click the ✏️ pencil (Edit) icon at the top right of the file
  5. Select all text (Ctrl+A / Cmd+A) and paste the new JSON (Ctrl+V / Cmd+V)
  6. Scroll down → click Commit changes
  7. Your live website updates in ~30 seconds ✅
